Project Overview

A back-end management system with RESTful api to keep track with every family's income and expense.

Tech Stack


Major Technologies: Python, Django, RESTful API, MySQL, Locust, Pytest, JWT

Django includes an admin page serving as a service desk, aiding users in managing safety concerns and sensitive deletions. Built on Django's Rest Framework and Joser JWT authentication, the platform ensures secure user access. Unauthorized users are restricted from modifying records, with data isolation preventing users from accessing other family's information. ORM, Python's automated and performance test (Locust) guarantee smooth performance under high user loads while minimize errors.

User Mannual

User Login and Authentication:

  1. Create a user through authentication endpoint by entering first name, last name, password and email: /auth/user

  2. After doing so, generate access token through /jwt/create/. Remember to save the token locally through front-end (eg. LocalStorage in Javascript) as it will be expired after a period of time (30 days for development sake)

    3. Note: Once a user is created, an associate member is created in our database for tracking money, link your family or view statistics

Family Creation:

Along with your JWT authentication header, you are autherized to create a family through: /tracker/families enpoints. Imput the family name for your family and a password. The password here is not for accessing a family, but for our service desk to check your identity. Instead, a family code will be generated and that will be used for future secure access and management of your own family.
